# Projekat - Aplikacija sa naprednim sistemom autentikacije

Ovaj projekat je razvijen u okviru predmeta **Internet tehnologije** i sastoji se iz dva dela:
- **Backend** (u folderu `seminarski`)
- **Frontend** (u folderu `Domaći2/blog`)

- ## ✅ Uputstvo za pokretanje projekta

### 🔹 KORAK 1: Kloniranje repozitorijuma
U command promptu pokrenuti:
git clone https://github.com/elab-development/internet-tehnologije-2024-projekat-appsanaprednomautentikacijom_2021_0462.git

### 🔹 KORAK 2: Instalacija zavisnosti
cd internet-tehnologije-2024-projekat-appsanaprednomautentikacijom_2021_0462

npm install

### 🔹 KORAK 3: Pokretanje backenda
cd seminarski

npm run start

### 🔹 KORAK 4: Pokretanje frontend-a (u novom terminalu)

Otvoriti novi prozor terminala (novi cmd) i pokrenuti sledeće komande:

cd internet-tehnologije-2024-projekat-appsanaprednomautentikacijom_2021_0462

cd Domaći2/blog

npm run start

- ## NAPOMENA

Potrebno je imati instaliran Node.js i npm

Backend i frontend treba da se pokrenu u odvojenim terminalima

- ## FUNKCIONALNOSTI

- Kreiranje naloga

- Potvrda naloga putem mejla

- Podešavanje dvofaktorske autentikacije

- Prijavljivanje (logovanje)

- Dvofaktorska autentikacija prilikom prijave

- Kreiranje postova

- Kreiranje komentara

- Brisanje postova

- Light/Dark mode
